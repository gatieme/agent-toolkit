## Codebase Patterns
- 项目使用 Python 3，py_compile 进行语法检查
- get_github_stars.py 使用阈值评分算法，_score_component() 实现核心逻辑
- GitHub 仓库路径格式为 owner/repo，通过 parse_repo_string() 验证
- 测试使用 pytest 和 unittest.mock 进行 Mock 测试
- update_plugin_readme.py 使用 tempfile + shutil 实现原子写入
